# 利用方法
##　マッチング機能
・テスト用ユーザーアカウントでログインを行う。

・ヘッダーの'マッチング'よりマッチング投稿一覧ページへ遷移する。

・一覧ページ上部の検索フィールドから、期間、メニュー、コメントの内容を検索できる。

・一覧ページの新規投稿ボタンをクリックして新規投稿ページへ遷移。全て入力して'投稿する'ボタンを押すと、一覧ページに遷移して投稿内容が表示される。（実装中）

・各投稿内容をクリックして詳細ページへ遷移できる。自分の投稿意外では編集や削除ボタンは表示されない。（実装中）

・一覧ページから自身の投稿詳細ページに遷移すると、編集と削除ボタンがある。（実装中）

・ヘッダー'ログアウト'ボタンからログアウトする。

・ログインページの'事業者の方はこちら'をクリックして、事業者としてログインする。

・ヘッダーの'マッチング'よりマッチング投稿一覧ページへ遷移する。

・任意の投稿をクリックして、投稿詳細ページへ遷移する。（実装中）

・詳細ページにある'マッチングする'ボタンをクリックする。（実装中）

・ユーザーとのチャットルームが作成される。（実装中）

・メッセージを投稿する。（実装中）

・ヘッダー'ログアウト'ボタンからログアウトする。

・テスト用ユーザーアカウントでログインを行う。

・ヘッダーのニックネームをクリックしてマイページに遷移する。（実装中）

・交渉中のマッチング一覧より該当のものをクリックしてチャットルームへ遷移する。（実装中）

・チャットルームで'マッチング成立'ボタンをクリックして、マッチングを成立させる。（TBD）

・支払い内容の入力を行い、支払いを実行する。（TBD）

・支払いが実行され、マッチングが成立する。（TBD）

#  アプリケーションを作成した背景
本アプリケーションが想定しているユーザーは20〜30代女性、地方在住。フリマアプリやSNS、マッチングアプリを生活の一部として利用している。一方で地方では生活インフラでもある車の利用において、日常のメンテナンスや法定点検においてもっと便利に利用できないかと思うことがある。しかしどこに整備をお願いして良いかわからないうえに、近所にある自動車整備工場は何となく入りにくいと感じている。

もう一方の想定ユーザーとして、自動車整備事業者である。一度来店してもらえればリピーターになってもらえる自信はあるものの、一見のお客様に来店いただく機会はほぼない。それならばと新規顧客を開拓したい思いはあるが、広告宣伝には多くのお金をかけられない。

本アプリケーションは整備をお願いしたい自動車ユーザーと新規顧客を獲得したい整備事業者をマッチングさせることで両者の課題を解決し新たな価値を提供するサービスである。
